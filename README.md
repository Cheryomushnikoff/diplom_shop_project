# Интернет-магазин с онлайн-оплатой (ЮKassa)
## Описание проекта

Данный проект представляет собой полнофункциональный интернет-магазин с системой онлайн-оплаты,
 личным кабинетом пользователя, управлением заказами и интеграцией с платёжным сервисом ЮKassa.

### Проект разработан в рамках дипломной работы и демонстрирует:

- клиент-серверную архитектуру
- работу REST API
- интеграцию с платёжной системой
- обработку webhook-событий
- систему возврата денежных средств
- разграничение прав доступа

## Архитектура проекта

`Проект построен по принципу разделения frontend и backend.`

### Backend:

- Python 3
- Django
- Django REST Framework
- PostgreSQL / SQLite
- ЮKassa SDK
- SMTP email

### Frontend:

- JavaScript
- React
- React Router
- Axios
- Bootstrap

## Реализованный функционал

### Пользователь:

- Регистрация и авторизация
- Реализация JWT
- Подтверждение email
- Личный кабинет
- История заказов
- Смена пароля

### Товары:
- Отзывы на товары
- Оставить отзыв можно только купив 
- Рейтинг товаров

### Заказы:

- Добавление товаров в корзину
- Оформление заказа
- Отмена заказа
- Автоматическое изменение статусов
- Просмотр деталей заказа

### Онлайн-оплата:

- Интеграция с ЮKassa (sandbox)
- Создание платежа
- Редирект на страницу оплаты
- Обработка webhook
- Подтверждение успешной оплаты

### Возврат средств:

- Возврат через ЮKassa API
- Проверка владельца заказа
- Обработка refund webhook
- Автоматическое обновление статуса

## Логика статусов заказа

```new → paid → processing → shipped → completed```

### При отмене:

```new → canceled```


### При возврате:

```paid → refunded```

## Установка и запуск

### Backend:
``` 
cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

### Frontend
```
cd frontend
npm install
npm run dev
```
